This project is a Full Stack E-Commerce Web Application with role-based authentication and advanced features like recommendation systems, analytics, and secure online payments.

The platform connects Customers, Suppliers, Delivery Personnel, and Admins to create a complete online shopping ecosystem.

## 👥 User Roles & Functionalities ##
## 👤 Customer
Register & Login
Browse and Search Products
View Product Details
Add to Cart
Place Orders
Secure Online Payment using eSewa Integration
View Order History
Get Product Recommendations
View Spending Analytics (Graph Visualization)
## 🏪 Supplier
Add, Update, Delete Products
Manage Product Inventory
View Orders
Assign Delivery Person to Orders
View Sales Analytics using Graphs
## 🚚 Delivery Person
View Assigned Orders
Update Delivery Status
Deliver Products to Customers
## 🛠️ Admin
Manage All Users (Customer, Supplier, Delivery)
Monitor Orders & Transactions
Manage Products & Categories
Control Overall System
## 🤖 Recommendation System

This project includes two types of recommendation systems:

1. Item-Based Collaborative Filtering
Uses Cosine Similarity
Recommends products based on user behavior (views, purchases)
2. Content-Based Filtering
Recommends products based on product attributes like:
Category
Description
Tags
## 📊 Data Visualization
Implemented using Pandas
Graphs for:
Customer Spending Behavior
Supplier Sales Performance

Integrated eSewa Payment Gateway
Ensures secure and reliable online transactions
